# TorchSig Dataset

**Source:** [TorchSig GitHub](https://github.com/torchdsp/torchsig)

## Description
Synthetic RF waveforms used for modulation classification and spectrum learning.

- ~5.3 million samples
- Size: ~340 GB total
- Formats: `.h5`, `.npy`, `.json`
- Includes multiple modulation types, SNR ranges
